

#  SmartConnect : AI Powered Mentor-Startup Relationship Management System

An AI-powered innovation ecosystem platform that automates relationships between startups and mentors using intelligent matching and structured ecosystem data.

Built to solve critical issues of innovation ecosystem platforms where :
- Manual & Inconsistent Coordination becomes common in managing growing data
- Hard to scale system act as the performance bottleneck
- Relationships between parties are not automated and intelligent

This makes scaling difficult and prevents reuse of past data for smarter decision-making.

---

Smart Connect introduces an AI-driven ecosystem intelligence system that:
- Extracts structured startup data from onboarding inputs
- Matches startups with the most suitable mentors using AI (Gemini)
- Creates reusable ecosystem relationships
- Tracks success metrics over time to improve future matching

---

##  System Architecture

Frontend (React / Vite)
- Dashboard + Ecosystem UI

Backend (Nodejs + Express)
- AI Matching Engine (Gemini)
- REST APIs
- Firebase & Firestore Database Integration

AI Models : Gemini from Google for reasoning, decision-making and data extraction

Cloud Database : (Firebase Firestore)
- startups
- mentors
- relationships

## Core AI Features

### 1. Startup Intelligence Extraction
- Converts pitch deck / form input into structured startup profiles

### 2. Mentor Matching AI
- Uses Gemini to evaluate:
  - Skills
  - Industry alignment
  - Experience relevance
- Outputs:
  - Best mentor
  - Match score
  - AI reasoning

### 3. Relationship Intelligence Layer
- Stores mentor-startup relationships
- Tracks success metrics:
  - Funding raised
  - Partnerships secured
  - Revenue growth

##  Setup Instructions

### Clone repository
`git clone https://github.com/imfiitriii/smart-connect.git`

### Backend setup
`cd backend`
`npm install`
`node server.js`

### Frontend setup
`cd frontend`
`npm install`
`npm run dev`
